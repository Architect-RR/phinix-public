## Repo 層級

- 已建立新的空 GitHub repo
- repo 名稱已確認

## 內容層級

- 不含 token / secret / bridge 憑證

- 不誇大未完成能力

## 文件層級

- README 已說清楚定位
- PUBLIC_SCOPE 已說清楚公開邊界

## 發布建議

建議第一版公開時，只先放：

- 這個 `github_public/` 目錄內的內容
- 經整理後的高層架構與抽象接口

不要第一版就放：

- 完整 private 主腦實作
- 具身敏感控制鏈
- 訓練資料
- vendor build 產物
