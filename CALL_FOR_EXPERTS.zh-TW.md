# Call for Experts

Languages: [English](CALL_FOR_EXPERTS.md) | **繁體中文**

PHINIX 正在往以下方向收斂：

- 本地主權認知核心
- 具身 AI companion 架構
- 主動式長時推理
- 未來人形機器人的 cognition layer

如果你在以下領域有經驗，你的回饋會非常有幫助。

## 我們想邀請的專家方向

### 1. Agents、memory、world models

我們想討論：

- 長時記憶如何與即時層分離
- world-state updates 如何避免自我污染與虛假確信
- 背景推理如何不拖垮 live responsiveness

### 2. Proactive AI 與 companion design

我們想討論：

- 什麼時候系統該主動提出結論
- 什麼時候應該保持安靜
- 什麼樣的提醒節奏像陪伴而不是噪音

### 3. AR、wearables、HCI

我們想討論：

- 波導或 HUD 的資訊密度要如何拿捏
- 語音與視覺互動如何混合
- hands-free 任務支援最佳實踐是什麼

### 4. Robotics 與 humanoid integration

我們想討論：

- 主腦和機器人身體應如何分離
- perception、planning、embodiment 的邊界如何切
- 同一個 cognition core 是否能跨多種機器人身體延續

### 5. Governance、safety、alignment

我們想討論：

- stuck issues 應如何升級
- proactive boundary 應如何設計
- 具身行動前應有什麼 safety gates
- 長期 integrity 要如何維持

## PHINIX 的核心問題

1. 如何讓主動性有用而不打擾？
2. 如何讓 companion system 在長時間使用中維持一致性？
3. 若未來接上人形機器人，哪些應屬於 cognition core，哪些應留在 body control？
4. 如何讓 local-first 架構保持可延展性，同時不削弱邊界？

## 參與方式

- 開 architecture issues
- 提交 docs 或 interfaces 相關 PR
- 提供高層設計回饋
- 協助定義 public/private split
